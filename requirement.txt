## Frontend Requirements

This project relies on the following front-end technologies and libraries:

- HTML5: Core structure of the website.
- CSS3**: Used for styling and layout, with custom styles in `styles.css`.
- JavaScript (ES6+): Handles the quiz logic and user interactions.
- Bootstrap 5.3.0: Used for responsive design.

Make sure to use modern browsers (Chrome, Firefox, Edge) for full compatibility.
